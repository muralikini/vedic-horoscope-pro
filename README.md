# 🌟 VedicHoroscope Pro

A beautiful, professional **Vedic Astrology Web App** with North/South Indian charts, Annual predictions, Gemstone recommendations, Vedic + Lal Kitab remedies.

Built with Streamlit.

## ✨ Features

- Clean, modern interface
- North Indian & South Indian chart styles
- Detailed Natal Chart with planetary positions
- Annual Horoscope for any year
- Gemstone analysis with wearing instructions
- Vedic + Lal Kitab remedies
- Download full PDF report
- Responsive design

## 🚀 How to Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/YOUR-USERNAME/vedic-horoscope-pro.git
   cd vedic-horoscope-pro
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the app:
   ```bash
   streamlit run vedic_horoscope_pro.py
   ```

## 🌐 Deploy on Streamlit Cloud (Free)

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Click **New app** → Connect your GitHub repo
4. Set main file as `vedic_horoscope_pro.py`
5. Deploy!

Your app will be live instantly.
